About Movie Review Classifer

The aim of this project is to make a movie review classifier using machine learning and deep learning and then use the same to predict rating of a movie. This project uses machine learning and deep learning to classify reviews of a movie as positive or negative (Sentiment Analysis) and then predict rating of that movie
